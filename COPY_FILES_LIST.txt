========================================
           CÁCH SỬ DỤNG
========================================

1️⃣ COPY FILES:
   - Tạo thư mục mới trên máy đích
   - Copy tất cả files "BẮT BUỘC" và "TÙY CHỌN"
   - Đảm bảo có đủ 3 file template .wav

2️⃣ CHẠY SETUP:
   - Double-click vào install_run.bat
   - Chọn [1] để chạy chương trình
   - Hoặc chọn [2] để build EXE

3️⃣ YÊU CẦU HỆ THỐNG:
   ✅ Windows 10/11
   ✅ Python 3.8+ (tự động hướng dẫn cài)
   ✅ Internet connection (để tải thư viện)
   ✅ Thiết bị GSM 32 cổng
   🆕 librosa + fastdtw (Professional MFCC analysis)
   🆕 scipy >= 1.10.1 (signal processing)

========================================
           LƯU Ý QUAN TRỌNG  
========================================

🔊 AUDIO SETUP:
   - Kết nối audio output của GSM → Line-in của máy tính
   - Hoặc sử dụng audio cable 3.5mm
   - Test microphone/line-in hoạt động

📡 GSM SETUP:
   - Thiết bị GSM kết nối COM35-COM66
   - ESP32 kết nối COM67 (nếu dùng)
   - Cài driver GSM devices

📝 FILE INPUT:
   - Chuẩn bị file .txt chứa danh sách số điện thoại
   - Mỗi số một dòng (ví dụ: 0123456789)
